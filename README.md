# Go-Pay Account Manager

## Features
- Check Balance
- Send Rp.1
- Send Rp.1 (Mass Sender)
- Send other amount
- Send other amount (Mass Sender)
- Add accounts
- Delete accounts

## Instalation
- `git clone https://github.com/corrykalam/gopay-manager`
- `pip3 install requests`

## RUN
- `python3 run.py`
